========================================
CUSTOMIZATION QUICK START
========================================

BEFORE LAUNCHING:

1. SET HOUSE LOCATION
   Open Settings section → Enter latitude & longitude
   Example: 12.9716, 77.5946 (Bangalore)
   
2. SET HOST PHONE
   Open Settings section → Enter phone number
   
3. REPLACE IMAGES
   Replace 6 images in /assets/images/
   Keep filenames: home1.jpg - home6.jpg
   
4. REPLACE VIDEO
   Replace /assets/videos/journey.mp4 with your video
   (MP4 format recommended)
   
5. CHANGE COLOR THEME
   Open Settings → Click color button (auto-saves)

SUPPORT & HELP
========================================

VOICE NOT WORKING?
→ Check microphone/speaker permissions
→ Try Chrome browser (best support)
→ Check internet connection
→ Refresh page

IMAGES NOT SHOWING?
→ Check /assets/images/ folder
→ Verify filenames (home1.jpg - home6.jpg)
→ Ensure JPG format

MAPS NOT OPENING?
→ Verify coordinates are correct
→ Check internet connection
→ Ensure browser allows popups

BLESSINGS NOT SAVING?
→ Check browser settings allow localStorage
→ Disable private/incognito mode
→ Clear browser cache
