⚙️ CONFIGURATION (5 MINUTES) - BEFORE LAUNCHING
════════════════════════════════════════════════════════════════════════════

REQUIRED CHANGES (3):

1. Set House Location
   • Open website in browser
   • Scroll to Settings (⚙️)
   • Under "📍 House Location"
   • Enter latitude (e.g., 12.9716)
   • Enter longitude (e.g., 77.5946)
   • Enter address (optional)
   • Click "💾 Save Location"

2. Set Host Phone
   • Still in Settings
   • Under "☎️ Host Contact"
   • Enter phone number (+91 format)
   • Click "💾 Save Phone"

3. Replace Images & Video
   • Go to /assets/images/
   • Replace home1.jpg - home6.jpg
   • Keep the same filenames!
   • Go to /assets/videos/
   • Replace journey.mp4
   • Use MP4 format

OPTIONAL CUSTOMIZATIONS:

• Change Color Theme
  └─ Click any color button in Settings

• Change Language
  └─ Select English/Kannada in Settings

• Enable Auto-Play Welcome
  └─ Check "Auto-play" in Settings

• Edit Welcome Message
  └─ Edit JavaScript (advanced)

🐛 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

Voice Not Working?
→ Check microphone/speaker in browser settings
→ Try Chrome (best voice support)
→ Refresh the page
→ Check internet connection

Images Not Showing?
→ Verify files in /assets/images/
→ Check filenames (home1.jpg - home6.jpg)
→ Try JPG instead of PNG format

Maps Not Opening?
→ Check internet connection
→ Verify coordinates are correct
→ Try different browser
→ Check if popups are blocked

Blessings Not Saving?
→ Enable localStorage in browser
→ Exit private/incognito mode
→ Clear browser cache
→ Try different browser
